Info

This is a CGTek copy of the companion app for Evothings Studio 2.0.

Use this app with the Evothings Studio 2.0 Workbench desktop application to develop mobile apps for IoT in HTML5/JavaScript. The desktop application is available at evothings.com/download.

How to use

Launch the Evothings Workbench desktop application. Follow the on screen instructions to get a connect key.

Enter the connect key into this app and tap the Connect button.

Click the Run button of a project in the Workbench to launch an app on this device.

Learn more

For more information and tutorials, visit evothings.com (note that tutorials are not yet updated for Evothings Studio 2.0, but all tutorial examples and JavaScript libraries are fully compatible with this app).
